1. Objective:

This script generates vCard files from a database containing contact information. Each vCard saved in .vcf format. 


2. Input:

csv file (employees list) added to database.  

python3 create_vcf.py initdb  - To initialize the database

python3 create_vcf.py import names.csv - Lqad the csv

python3 create_vcf.py query 11  - To get the vcard of an single employee.

   
3. Output:


Generate one vCard file per row in the database.

 Sample vcard file -

    BEGIN:VCARD
    VERSION:2.1
    N:Bennett;Amber
    FN:Amber Bennett
    ORG:Authors, Inc.
    TITLE:Bonds trader
    TEL;WORK;VOICE:911.815.0504
    ADR;WORK:;;100 Flat Grape Dr.;Fresno;CA;95555;United States of America
    EMAIL;PREF;INTERNET:amber.benne@bailey.biz
    REV:20150922T195243Z
    END:VCARD

The vCard file will have name as file name.
